# 5.3 查看AI的记忆

---

上一节，你学会了让 OpenClaw 记住你的偏好和重要事项。

现在你可能会想：**它到底记了什么？我怎么知道它记对了没有？**

这就是 OpenClaw 记忆系统的一个核心特点：**完全透明。**

这一节，我们用3分钟，学会查看和管理记忆。

---

## 记忆文件在哪？

在你的工作区目录下（`~/.openclaw/workspace/`）：

```
├── MEMORY.md              # 档案柜：长期记忆
└── memory/
    ├── 2026-03-08.md      # 今天的日记
    ├── 2026-03-07.md      # 昨天的日记
    └── ...
```

**这些都是普通的 Markdown 文件。** 用任何文本编辑器都能打开。

**记忆完全透明。** 随时打开查看，发现记错了直接改，记了太多没用的直接删。

---

## 试一试：查看记忆内容

直接在聊天里问：

> ```
> 你记住了关于我的哪些信息？
> ```

它会列出记住的内容：

> 我记得关于你的信息：
> - 偏好：回复简洁，不啰嗦
> - 习惯：早上9点前不要打扰
> - 重要事项：下周三交项目报告
> - 联系人：客户李四，邮箱 lisi@company.com

**你也可以直接打开 MEMORY.md 文件查看。**

---

## 试一试：修改记错的内容

如果它记错了，直接告诉它：

> ```
> 我喝美式，不是拿铁，帮我改一下
> ```

它会自动修改：

> 已修改：咖啡偏好 → 美式

**你也可以自己打开文件改。** 就是普通的文本，想改什么改什么。

---

## 试一试：搜索之前的记忆

想找之前说过的事：

> ```
> 搜索记忆中关于客户的内容
> ```

它会搜索所有记忆文件，找出相关内容：

> 找到 3 条相关记录：
> - 客户李四，邮箱 lisi@company.com
> - 客户张三提到下周要开会
> - 客户反馈产品A有个bug

**不只是关键词匹配。** 你说“交通工具”，它能找到“汽车”、“自行车”——意思相近的都能找到。

---

## 试一试：删除不想保留的内容

两种方式：

**方式一：直接告诉它**

> ```
> 把关于咖啡的偏好删掉，不用记这个
> ```

**方式二：手动删除**

打开 MEMORY.md，找到那一行，删掉，保存。

---

## 记忆文件要备份吗？

这些是普通文件，你想怎么备份都行：
- Git 提交到私有仓库
- 同步到云盘
- 复制到 U 盘

**数据在你自己手里。**

---

## 这一节，你做了什么

| 学了什么 | 核心理解 |
|----------|----------|
| 查看记忆 | 问它“你记住了什么”或打开文件 |
| 修改记忆 | 直接告诉它或手动编辑 |
| 搜索记忆 | 说“搜索记忆中关于...” |
| 删除记忆 | 告诉它删或手动删 |

关键是：**记忆完全透明，你随时可以查看和修改。**
