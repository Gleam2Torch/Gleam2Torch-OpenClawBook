# 3.3 验证部署成功

---

上一节，你运行了 `openclaw start`，看到 Gateway 启动的提示。

这一节，我们来验证一下：**OpenClaw 真的能用了吗？**

---

## 第一步：打开聊天界面

在浏览器地址栏输入：

```
http://127.0.0.1:18789/chat
```

按回车。

如果你看到一个聊天界面，说明 **Web UI 正常运行**。

---

## 第二步：确认OpenClaw已启动

首次进入聊天界面，OpenClaw会主动问候：

> 你好！我刚醒来，这里是全新的工作空间。我们来认识一下吧？

看到这条消息，说明 **OpenClaw 已经正常运行**。

---

## 部署完成！

恭喜你，OpenClaw 已经部署成功。

现在，你可以开始和它对话了。
