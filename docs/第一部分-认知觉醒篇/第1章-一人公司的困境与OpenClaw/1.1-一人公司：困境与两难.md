# 1.1 一人公司：困境与两难

---

凌晨两点，我还在回消息。

这个场景，我太熟悉了。

白天上班，晚上做副业。客户的消息要回，项目要推进，邮件要处理……

我以为这就是一个人的常态。

后来我发现，不止我一个人这样。

你可能也一样。

---

说实话，我不知道你现在的状态。

也许是早上6点，闹钟还没响，手机就亮了。

客户问你："那个订单发货了吗？"供应商催你："款项什么时候结？"

你迷迷糊糊回了消息，想再睡会儿，但脑子已经清醒了。

也许是凌晨两点，手机屏幕的光映着你疲惫的脸。

你刚刚回完最后一个客户的消息，本来想睡了，但又想起明天要发的版本还有一个Bug没修。

也许是周末的下午，窗外在下雨。

你本来答应陪孩子去公园，但那个运营活动的文案还没写完，社群里还有人@你问问题。

也许是刚发完工资的那天，你看着银行卡余额，脑子里盘算着：

"要是能多一个人帮忙就好了。"

然后你打开招聘软件，看了一眼运营助理的薪资——一线城市15000-25000元/月，二三线城市也要10000-18000元/月，还要交社保公积金，还要办公场地……

你默默关掉了页面。

**"算了，还是自己扛吧。"**

这可能是你对自己说的最多的一句话。

---

## 一个人，就是一家公司

一人公司是什么？

不是只有一个人在工作。

而是：**一个人扛了一整个公司该扛的所有事。**

你可能有不同的身份标签：

- 独立开发者，白天写代码，晚上回客服，周末做运营
- 小红书博主，每天要更新笔记、回复评论、对接品牌方
- 抖音创作者，拍视频、剪片子、盯数据、回私信
- 淘宝店主，选品、上架、客服、发货、管财务、做活动
- 自媒体创作者，写文章、剪视频、回复评论、对接商务
- 咨询顾问，做方案、谈客户、写文档、管账目

不管你是哪一种，本质都一样：

**你一个人 = 产品 + 运营 + 客服 + 财务 + 所有的其他事情。**

说个真实的故事。

**先说小张，他是个独立开发者，做了一个效率工具App。**

他的每天大概是这样的：

| 时间段 | 正在做什么 |
|--------|-----------|
| 9:00-10:00 | 回复昨晚积累的客服消息 |
| 10:00-12:00 | 写代码（中间被打断3次回客服） |
| 12:00-13:00 | 午休（手机一直在响） |
| 14:00-15:00 | 处理用户反馈，修复紧急Bug |
| 15:00-16:00 | 写公众号推文 |
| 16:00-17:00 | 对接商务合作 |
| 19:00-21:00 | 继续写代码 |
| 21:00-23:00 | 回复社群消息，准备明天的内容 |

你想专心做产品，但70%的时间花在了琐事上。

你想写出更好的代码，但总是在被打断。

你想休息一天，但事情永远做不完。

**你不是不努力，你是太忙了。**

**再说小李，她是个小红书博主，主要分享穿搭和美妆。**

她的每天大概是这样的：

| 时间段 | 正在做什么 |
|--------|-----------|
| 8:00-9:00 | 回复昨晚的私信和评论 |
| 9:00-11:00 | 拍摄今天的穿搭照片（找角度、换衣服、修图） |
| 11:00-12:00 | 写笔记文案，研究话题标签 |
| 12:00-13:00 | 吃饭（手机不离手，刷竞品内容） |
| 14:00-16:00 | 剪视频、修图、准备发布 |
| 16:00-18:00 | 对接品牌方，谈合作方案 |
| 19:00-21:00 | 盯数据、回评论、维护社群 |
| 21:00-23:00 | 策划明天的内容、选品、研究爆款 |

你想做出更好的内容，但80%的时间花在了拍摄、剪辑、回复消息上。

你想休息一天，但一天不更新数据就掉，粉丝就流失。

**你不是不努力，你是被琐事淹没了。**

---

## 时间永远不够用

一人公司最可怕的不是事情多，而是：**你不知道时间去哪了。**

早上起来，列了满满一张待办清单。

晚上躺下，想不起来今天到底做了什么有意义的事。

好像一直在处理别人的事情，一直在"救火"，一直在应付。

自己的产品？自己的节奏？自己的时间？

不存在。

**时间就像沙子，你抓得越紧，流失得越快。**

---

## 想雇人，但一算账就放弃了

你肯定想过雇人帮忙。

谁不想呢？

雇一个运营助理，客服有人回了，内容有人写了，社群有人管了……

你终于可以专心做你认为有价值的事。

每天醒来，不再是那一堆未读消息。

周末可以真的休息，不用担心错过重要的事情。

晚上可以早点睡，不用再熬到凌晨。

这就是雇人的理想状态。

**然后你打开了招聘软件。**

你输入"运营助理"，跳出来一堆简历。

你看了一眼薪资要求——

一线城市：7000-11000元/月

二三线城市：5000-8000元/月

还不止这些。

| 成本项 | 金额（月） |
|--------|-----------|
| 运营助理 | 7000-11000元 |
| 客服专员 | 5500-8500元 |
| 社保公积金 | 1500-2500元 |
| 办公场地 | 1000-3000元 |
| **合计** | **15000-25000元/月** |

这是显性成本。

还有隐性成本。

招聘要2-4周，培训要40小时，管理每天至少1小时。

而且人不是机器。

会生病，会请假，会情绪低落，会离职。

**你花了时间、精力、金钱，最后可能还是要自己扛。**

---

## 雇人也有代价

Peter Steinberger，OpenClaw的创始人，曾经也走过这条路。

他做了一个年入百万美元的产品，雇了团队。

但最终他说：

> "那条路我走过，很无聊。如果你的想法是"拼命工作然后退休"，我不推荐。"

商业成功不代表没有职业倦怠。

即使雇了团队，**管理本身也是一件耗费精力的事**——你要操心招聘、培训、激励、冲突、离职……

**这哪里是找人帮忙，分明是给自己找了个新工作。**

---

## 雇人就像买彩票

我说这句话可能有点夸张，但你想想：

你花了15000-25000元/月，但你不确定能不能得到你想要的结果。

可能招的人不合适，你要重新招。

可能招的人很合适，但干两个月就走了。

可能招的人很合适也很稳定，但你需要花大量时间管理。

雇人的成本和风险都比较高。

而自己扛呢？至少你清楚会发生什么。

这就是为什么很多人宁愿自己扛——不是因为不想雇人，而是因为风险和成本都太大。

---

## 自己扛的代价

但自己扛，真的没有代价吗？

当然有。

而且代价可能比你想象的更大。

**身体的代价：**

你熬夜，你错过饭点，你没有运动，你睡眠不足。

你以为年轻能扛，但身体会记得一切。

**心理的代价：**

你焦虑，你压力大，你情绪低落。

你觉得自己好像在做所有事，但又好像什么都没做好。

**效率的代价：**

你想专心做产品，但70%的时间花在琐事上。

你想写出更好的代码，但总是在被打断。

你想做出更好的内容，但总是在回复消息。

**机会的代价：**

这可能最重要。

**你在琐事上花的时间，就是你没有花在重要事情上的时间。**

那些能让你的业务真正增长的事情。

那些能让你的产品变得更好的事情。

那些能让你感到有成就感的事情。

自己扛的问题在于：你能走，但走不快；能前进，但走不远。
