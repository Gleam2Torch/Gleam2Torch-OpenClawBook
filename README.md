## 这本书讲什么

一个叫 OpenClaw 的开源 AI Agent 框架，以及它如何成为你的"数字分身"。

**和 ChatGPT、Claude 这些传统 AI 不同：**

| 传统 AI | OpenClaw |
|---------|----------|
| 你问它才答 | 主动帮你做事 |
| 对话需要手动开启 | 24小时在线值守 |
| 只能陪你聊天 | 能操作你的电脑 |
| 每次都是"第一次见你" | 能记住重要信息 |

**它能帮你：**

- 记住你说的话，在合适的时候提醒你
- 主动整理文件、处理消息、生成报告
- 在你睡觉的时候，帮你盯着重要的事情
- 接入飞书、钉钉、企业微信，成为你的全天候助理

**它不是替代你，而是成为你的延伸。**
